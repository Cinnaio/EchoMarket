# 📜 EchoMarket 全球市场指南

欢迎来到 **EchoMarket** 的世界！这是一个现代化的自由贸易系统，致力于让每一位冒险者都能拥有专属的 NPC 商店。告别传统的箱子商店，在这里，你可以雇佣一名专属的 NPC 代理人，为你全天候打理生意，向全世界展示你的珍宝。

---

## 🌟 核心概念 (Core Concepts)

在开始商业帝国之前，请熟悉以下三个概念：

1.  **商店 (Shop)**: 你的实体摊位，由一名 NPC（非玩家角色）代表。玩家可以在世界中直接点击它来购买商品，你也可以通过它管理库存。
2.  **全球市场 (Global Market)**: 一个远程的交易界面。无论身在何处，所有玩家都可以通过指令打开这个菜单，浏览全服的商店列表。
3.  **留言板 (Board)**: 位于全球市场中心的广告牌。你可以在这里发布悬赏、广告或求购信息，让全服玩家都能看到。

---

## 🏪 第一章：开设商店 (Entrepreneurship)

### 1. 选址
首先，你需要找一块风水宝地。
*   确保你面前（5格内）有空地。
*   **注意**: 你不能在受保护的区域（如 WorldGuard 禁止建造的区域）开设商店，除非你拥有特殊权限。

### 2. 雇佣代理人
看着你选好的位置，输入指令：
```
/market create [商店名称]
```
*例如：`/market create 宝藏屋`*

> **✅ 成功后**: 一个以你为原型的 NPC 会立刻出现在你面前，它就是你的专属店员！

---

## 💰 第二章：管理经营 (Management)

拥有了商店，接下来就是上架商品和装修门面。

### 1. 上架商品
无需复杂的箱子操作，只需简单的指令：
1.  将你要出售的物品拿在 **主手**。
2.  输入指令设定价格：
    ```
    /market sell <价格>
    ```
    *例如：`/market sell 100` 表示将手中的物品以 100 金币的价格上架。*
3.  *物品会立即从你手中消失，进入商店的无限货架。*

### 2. 装修门面
一个好的形象能吸引更多顾客：
*   **改名**: `/market name <新名称>`
    *   *给商店起个响亮的名字。*
*   **简介**: `/market desc <介绍语>`
    *   *写一段吸引人的标语，会显示在列表详情中。*
*   **换肤**: `/market skin <玩家名>`
    *   *让你的 NPC 穿上其他玩家的皮肤（例如 `Notch` 或 `MHF_Villager`）。*

### 3. 移除商店
如果你想关店歇业或通过搬迁重开：
*   看着你的 NPC，输入 `/market remove`。
*   或者使用 `/market remove <ID>` 远程删除。

## 🛍️ 第三章：购物消费 (Shopping)

### 1. 浏览市场
无需跑图，随时随地查看全服商品：
```
/market
```
*   这会打开**全球市场菜单**，展示所有玩家的商店。
*   将鼠标悬停在头像上，可以看到商店的简介、位置和热度。

### 2. 购买物品
*   **🈲 禁止远程购买**: 为了保障后续跑商等其他多种玩法，限制交易仅在NPC周围4格范围内进行！
*   **实体店购买**: 如果你在路上遇到了其他玩家的商店 NPC，直接 **右键点击** 它也可以打开购买界面。

## 📜 第四章：全服留言 (Social)

想让你的商店瞬间爆火？去留言板打个广告吧！

1.  输入 `/market` 打开主菜单。
2.  点击底部的 **木牌图标** (全服留言板)。
3.  点击 **“发布留言”** 按钮，然后在聊天栏输入你的广告词。
    *   *你的留言将展示给全服所有打开市场的玩家！*

## ❓ 常见问题 (FAQ)

**Q: 我可以开多少个商店？**
A: 这取决于你在服务器的权限等级。普通玩家通常有限制，升级你的等级可能获得更多名额。

**Q: 为什么提示“离商店太远”？**
A: 上架商品 (`/market sell`) 需要你处于自己的商店附近。这是为了防止远程误操作。

**Q: 赚到的钱去哪了？**
A: 所有的交易都是即时到账的！一旦有人购买你的商品，金币会直接存入你的账户。

**Q: 我可以修改商店 NPC 的位置吗？**
A: 可以！你可以使用 `/market movehere [序号]` 指令将指定的商店直接移动到你当前站立的位置。

## ⌨️ 指令速查表 (Commands)

| 指令 (Command) | 参数 | 说明 |
| :--- | :--- | :--- |
| `/market` | 无 | 打开全球市场主菜单 |
| `/market create` | `[名称]` | 在准星位置创建商店 |
| `/market sell` | `<价格>` | 将手中物品上架出售 |
| `/market list` | 无 | 查看你拥有的所有商店 |
| `/market name` | `<名称>` | 修改当前商店的名称 |
| `/market desc` | `<介绍>` | 修改当前商店的简介 |
| `/market skin` | `<玩家名>` | 修改当前商店 NPC 的皮肤 |
| `/market movehere` | `[序号]` | 将商店移动到你当前站立的位置 |
| `/market remove` | `[ID]` | 删除商店（需看着 NPC 或指定 ID） |
