# 🛒 EchoMarket 玩家指南

欢迎使用 **EchoMarket** 全球市场系统！本指南将帮助你快速学会如何开设自己的商店、买卖物品以及赚取财富。

---

## 1. 浏览与购买 (我是买家)

### 🌍 如何找到商店？
你有两种方式查看服务器内的商店：
1. **命令菜单**: 输入 `/market` 打开全球商店列表菜单。点击任意玩家头像即可进入其商店。
2. **寻找 NPC**: 在游戏世界中，你会看到代表商店的 **NPC**（非玩家角色）。直接**右键点击 NPC** 即可打开该商店。

### 💰 如何购买物品？
1. 打开商店界面后，点击你感兴趣的物品。
2. 进入**购买确认页**：
   - <font color="red">🟥 红色玻璃</font>: 减少购买数量 (-1, -10, -64)
   - <font color="green">🟩 绿色玻璃</font>: 增加购买数量 (+1, +10, +64)
   - 💎 **钻石块**: 选择购买当前所有库存
   - ❇️ **绿宝石块**: **确认购买**
3. 点击绿宝石块后，钱将自动扣除，物品会进入你的背包。
   - *注意：如果背包满了，多余的物品会掉落在你脚下，请及时拾取！*

## 2. 开设商店 (我是店主)

想要拥有自己的铺子？只需简单几步：

### 🏪 创建商店
1. 找一个空旷、显眼的地方（例如主城广场或你的家门口）。
2. 输入命令：
   ```
   /market create [商店名称]
   ```
   *例如：`/market create 精品杂货铺`*
3. 一个属于你的 NPC 就会生成在你脚下！

### 📦 上架商品 (出售)
1. **必须站在你的商店 NPC 附近**（不能远程上架）。
2. **手持**你想要出售的物品。
3. 输入命令：
   ```
   /market sell <单价>
   ```
   *例如：`/market sell 50` 表示你手里的物品**每一个**卖 50 元。*
4. 系统会提示上架成功，并告知扣除的预计税费。

### 🎨 装修你的商店
你可以随时修改商店的信息，让它更吸引人：
*   **改名**: `/market name <新名称>`
*   **改描述**: `/market desc <一段吸引人的介绍>`
*   **换皮肤**: `/market skin <玩家ID>`
    *   *觉得 NPC 默认皮肤不好看？输入 `/market skin Notch` 试试！*

### ❌ 下架/取消商品
1. 打开你自己的商店界面。
2. **右键点击**你想要取回的商品。
3. 物品将退回到你的背包。
   * *注意：根据服务器设置，取消订单可能会收取少量违约金（手续费）。*

## 3. 常见问题 (FAQ)

**Q: 为什么我输入 `/market sell` 提示太远了？**
A: 为了防止滥用，你必须亲自站在你的商店 NPC 旁边（通常是 4 格范围内）才能上架货物。

**Q: 手续费是怎么算的？**
A: 
*   **交易税**: 当你的物品**成功卖出**时，系统会自动从货款中扣除一定比例（默认约 3%）。
*   **特殊税率**: 部分珍贵物品可能有更高的税率，上架时请留意提示。

**Q: 我能开几个店？**
A: 这取决于你的权限等级。普通玩家通常限制开设 1 个商店。

**Q: 怎么删除商店？**
A: 如果不想开了，可以对着你的 NPC 输入 `/market remove`，或者输入 `/market remove <商店ID>`。删除前请确保架上商品已全部取回！
